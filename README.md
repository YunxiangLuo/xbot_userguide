# 产品在线文档仓库

本项目为重德智能产品的在线文档仓库，包含xbot-u系列产品，xbot-q系列产品，xbot-arm机械臂等产品的用户使用手册和出场配置手册等内容。

```

├── xbotarm                         #xbotarm系列

│   └── single                      #单臂
├── xbotq                              #四轮小车

```
